MOBA - A Versatile Discord Management and Entertainment Bot

Discord.py

Sqlite3.py

Python

MOBA is a versatile Discord bot equipped with SQLite3-supported logging to maintain activity records efficiently. It handles commands smoothly through Discord’s slash command interface and features robust error management to ensure reliable functionality. One of its standout features is a built-in Taboo game, offering an interactive and entertaining experience for server members. MOBA provides a streamlined and responsive tool for server management and gaming fun.
